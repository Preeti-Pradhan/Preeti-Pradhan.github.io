HOTEL MANAGEMENT WEBSITE

The project , Hotel Management System is a website that allows the hotel manager to handle all activities online. HOSPITAL_MANAGEMENT_WEBSITE

The system incorporates various factors needed for smooth working and management of a hospital. This hospital management system allows administrator to store and manage various products and related data. This includes the beds/normal accomodation as well as patient data which includes name, number, address and various health parameters. Doctors can view patients treatment records and details easily. Users must register and login themselves first. I developed this website using wordpress.org. The system is flexible and convenient to used.
